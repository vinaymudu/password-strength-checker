# 🔐 Password Strength Checker

A Python-based cybersecurity tool that analyses password strength
and saves results to a file.

---

## 👨‍💻 Built By
Vinay — Cybersecurity & Python Enthusiast

---

## 📌 What This Tool Does
- Takes a password as input from the user
- Checks length, uppercase, numbers and special characters
- Rates the password as Strong, Medium, Weak or Very Weak
- Saves all results to a file called password_results.txt
- Handles errors gracefully without crashing

---

## 🚀 How To Run
1. Make sure Python is installed on your computer
2. Clone this repository
3. Open terminal and run:
   python password_checker.py
4. Enter any password to check its strength
5. Type 'quit' to exit

---

## 📊 How Passwords Are Rated

| Score | Rating |
|---|---|
| 4/4 | 🔥 Strong |
| 3/4 | ⚠️ Medium |
| 2/4 | ⚠️ Weak |
| 0-1/4 | ❌ Very Weak |

---

## ✅ Password Requirements Checked
- Minimum 16 characters
- At least one uppercase letter
- At least one number
- At least one special character (!@#$%^&*)
